Adding an Image Hover

If you took a look at the image gallery above, you most likely would have noticed that there was a slight effect when hovering over the images. The border changed color, style, and there was a shadow added. Not only that but the font changed colors and the photo becomes somewhat opaque. This was a relatively easy feature to implement into the site. Below is a snippet of code that shows what the structure of the image looks like for the image gallery.

As you can see above, there are different tags that are allowed to be used when making an image. Encompassing the entire image and caption is a figure tag, in side that tag is an image tag as well as a figcaption tag. The figure tag is used to hold the figure together. It kind of acts like a div tag when used in this sense. The image tag is simply just the image to be displayed on the site. Following that is the figcatpion; the caption for the image. Now that the formatting of the gallery figures is out of the way we can get to how the actual hover effect works. There is another snippet of code attached below that shows what the style of the gallery images will look like before the hover effect.

If you want to show a change from one state to another there obviously has to be an original state for that object. In this case there is an original border style and color, as well as a text color. In the css code, you can see that there is an original border style of solid and that it is also sky blue. The original font color is a dark slate grey. All these styling choices were based off what the website looked like previously. But I wanted to change that up when someone hovered over an image. I wanted the images to stand out, and change subtly. This is where the hover effect comes into play, I got to play around with many different things when I wanted to have an image hover effect. I did not want to take away from the pictures themselves but merely emphasize the one you were looking at in particular. Below is the snippet of code that allowed me to change the states when hovering over an image.

As you can see from the code above, I change some minor things when the image is hovered over. First off to get the effect of something when it is hovered over, you need to make sure that that even is listened for. In order to do that you need to put the specific element you want the user to hover over colon hover. This is what you see in yellow at the top of the code. Inside those brackets will execute when the user hovers over that element in particular. As you can see I put a hover effect on the entire figure, this includes the image and the caption as stated in the first paragraph. Once the user hovers over any part of the figure element, the opacity will change slightly. This adds a nice emphasis on the figure being hovered over. The next change that occurs is the border, instead of it being one solid border, it is now two separate lines, and it changes to the color orange. Then I put a box shadow around the border to make the figure pop off the page. Lastly, I changed the font color to dark orange to make it change completely. Now that you have read this entire blog on how the hover effect works on the image gallery, why don’t you go back up and try it for yourself. Try to think of everything that is going on when you hover over the image. Don’t just try to hover over the image itself, hover over the image caption or maybe just the border itself. What happens?
